# image-studio-streamlit
## Project Summary

AI Image Studio is an interactive web application developed using Python and Streamlit that enables users to generate high-quality AI images from text prompts using the Pollinations AI API. The application provides an intuitive and user-friendly interface where users can describe the image they want to create, choose from multiple artistic styles, customize image dimensions, and generate visually appealing AI artwork within seconds.

The project includes several features designed to enhance the user experience, such as Magic Enhance for improving prompt quality, customizable image width and height, reference image upload for visual guidance, multiple AI art styles, instant image preview, and one-click PNG image download. The application also maintains a session-based prompt history, allowing users to review previously generated prompts during the current session.

This project demonstrates practical knowledge of Python programming, Streamlit web application development, REST API integration, user interface design, session state management, file handling, and prompt engineering. It was developed as part of the MirAI School of Technology Virtual Summer Internship 2026 and showcases the ability to build interactive AI-powered applications with a focus on usability and modern web development practices.
